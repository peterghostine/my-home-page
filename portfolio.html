Portfolio

The People's Tournament

The People's Tournament — This fully-responsive site was built using HTML, CSS, and JavaScript. It features a mobile-friendly menu, displays a Facebook photo album, and offers social-sharing buttons. Event listeners were created on the buttons to enable users to view the event flyers in a modal window. The site is also A11Y-friendly.

Outmatch USA

Outmatch USA — This fully responsive storefront was built using WordPress and WooCommerce. It features smart coupons, social sharing, and payment processing via PayPal, Stripe, and Coinbase. Other third-party plug-ins were also used for the contact form and FAQ page, and to track abandoned carts.

LA Fitness

LA Fitness — LA Fitness's actual website makes is built using Bootstrap, but I reproduced it here using only Flex. I also made better use of white space, and I improved the site's responsiveness as the browser is resized from desktop all the way down to mobile.

Mixed Messages

Mixed Messages — This JavaScript project greets the user with three random pieces of information: a random Yogi Berra quote, a random adage, and a randomly-generated Powerball pick.
